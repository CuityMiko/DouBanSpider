# DouBanSpider
# 一个豆瓣读书的爬虫

爬取豆瓣读书上的每一个热门标签的第一页的图书，有图书名字，图书出版社信息，作者，评论数等，然后把信息存储到本地的一个CSV文件。

> 使用Python3

使用的库第三方库有

* requests
* bs4

> 运行
> python3 DouBanSpider.py
